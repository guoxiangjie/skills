## 变更记录格式说明

### 日期标题
- 格式：`## YYYY-MM-DD`
- 按时间倒序排列（最新的在上面）

### PRD 文件标识
- 更新文件：`### 📄 文件名 (旧版本 → 新版本)`
- 创建文件：`### 📄 文件名 (创建 v1.0)`
- 删除文件：`### 📄 文件名 (删除 v1.0)`

### 变更详情表格

| 字段 | 说明 |
|------|------|
| 变更项 | 具体变更的功能点或内容 |
| 变更前 | 变更之前的内容 |
| 变更后 | 变更之后的内容 |
| 变更原因 | 为什么做这个变更 |

详细格式规范请参考：[changelog_format.md](.trae/skills/prd-suite/templates/changelog_format.md)
