## 变更记录格式说明

### 日期标题
- 格式：`## YYYY-MM-DD`
- 按时间倒序排列（最新的在上面）

### 变更类型标识

| 类型 | 标识 | 含义 |
|------|------|------|
| 创建 | 🟢创建 | 新增 PRD 文档 |
| 更新 | 🟡更新 | 已有文档内容修改 |
| 删除 | 🔴删除 | 文档被移除 |

### 文件标识格式
```
### 📄 文件名 变更类型标识
```

示例：
- `### 📄 用户管理系统_main_prd.md 🟢创建`
- `### 📄 用户管理_module.md 🟡更新`
- `### 📄 废弃功能_module.md 🔴删除`

### 变更详情

记录变更涉及的章节及内容摘要：

```
#### 变更章节：章节名称

**变更内容**：简要描述该章节发生了什么变化
```

多条变更时按章节依次列出。
