# init 子命令

## 功能说明

初始化 PRD 文档包结构，创建完整的目录和文件。

## 工作流程

### 步骤1：询问生成位置
使用 AskUserQuestion 工具询问用户要在哪个位置生成 PRD 文档包。

### 步骤2：检查位置是否存在 PRD 文档包
- 检查目标位置下是否存在 `prd` 文件夹
- 如果存在，提示用户该位置已有 PRD 文档包，询问是否继续或选择其他位置
- 如果不存在，继续执行

### 步骤3：创建目录结构
在目标位置创建以下目录和文件：

```
目标位置/
└── prd/
    ├── README.md              # 项目说明文档
    ├── PRD_INDEX.md           # PRD模块总集索引
    ├── CHANGELOG.md           # 版本变更日志
    ├── main_prd.md            # 主PRD文档（空文件）
    └── modules/               # 功能模块PRD目录（空文件夹）
```

### 步骤4：生成文件内容
使用模板文件生成各个文档的内容，替换变量。

### 步骤5：确认完成
向用户确认 PRD 文档包已成功创建，并提示下一步可以使用 `prd-suite` skill 来生成具体内容。

## 注意事项

- 如果目标位置不存在，会自动创建
- 所有文件使用 UTF-8 编码
- `{{CURRENT_DATE}}` 替换为当前日期，格式为 YYYY-MM-DD
- `{{PROJECT_NAME}}` 替换为目标文件夹的名称
