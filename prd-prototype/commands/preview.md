# preview 命令 - 预览和导出原型

## 功能说明

预览已生成的原型项目，并提供导出功能。

## 工作流程

### 步骤 1：检查原型目录

检查项目目录下是否存在原型目录：
- 默认路径：`prd/prototype/`
- 检查 `index.html` 文件是否存在

**如果没有找到原型**：
> ⚠️ 未找到原型项目。请先使用 `/prd-prototype generate` 命令生成原型。

**路径确认**：
```
📂 原型目录：prd/prototype/
```

### 步骤 2：选择预览方式

提供预览选项：

| 选项 | 预览方式 | 说明 |
|------|----------|------|
| 1 | **浏览器打开** | 在系统默认浏览器中打开 index.html |
| 2 | **VS Code Live Preview** | 在 VS Code 中使用 Live Server |
| 3 | **Python HTTP Server** | 启动简易 HTTP 服务器 |

### 步骤 3：打开原型

**浏览器打开**：
```bash
open prd/prototype/index.html
```

**Python HTTP Server**（推荐，支持跳转）：
```bash
cd prd/prototype
python -m http.server 8080
# 然后打开 http://localhost:8080
```

### 步骤 4：导出选项

提供导出选项：

| 选项 | 导出方式 | 说明 |
|------|----------|------|
| 1 | **ZIP 打包** | 将原型目录打包为.zip 文件 |
| 2 | **复制到指定位置** | 复制原型到用户指定的路径 |
| 3 | **生成分享链接** | 如需线上分享，提供部署建议 |

### 步骤 5：完成预览

预览完成后：
- 如启动 HTTP 服务器，提示如何停止
- 如导出文件，确认导出位置

## 注意事项

- 静态 HTML 文件直接打开可能无法正常工作（部分跳转可能受限）
- 推荐使用 HTTP 服务器方式预览
- 导出前确保所有文件已保存
