# delete 子命令

## 功能说明

删除 PRD 文档，支持删除模块PRD。

## 子命令

### delete module
- **功能**：删除模块PRD文档
- **描述**：删除指定的模块PRD文档，并更新相关索引
- **支持**：安全确认和索引更新

## 工作流程

### 删除模块PRD流程

1. **选择模块**：选择要删除的模块PRD
2. **安全确认**：询问用户是否确认删除
3. **删除文件**：删除模块PRD文件
4. **更新索引**：更新 `PRD_INDEX.md`，移除该模块的记录
5. **更新变更日志**：在 `CHANGELOG.md` 中记录删除操作
6. **确认完成**：向用户确认删除完成

## 注意事项

- 只支持删除模块PRD，不支持删除主PRD
- 删除前会进行安全确认
- 删除后会自动更新相关索引和变更日志
- 无法恢复已删除的文档，请谨慎操作
