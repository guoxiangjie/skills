# 模块 PRD 问答模式询问模板

## 概述

本文档提供了问答模式下每个章节的询问问题模板、参考信息提取规则和章节生成指导说明。

## 第 1 章：文档基础信息

### 处理方式
**自动填充，无需询问**

### 自动提取信息
- **文档名称**：`{模块名称} 产品需求文档`
- **作者**：从主 PRD 中提取
- **创建时间**：当前日期
- **当前版本**：v1.0
- **关联主PRD**：主 PRD 名称及版本

## 章节审核选项

每个章节生成后，提供以下审核选项：

### 标准选项
- ✅ **确认通过**：继续下一章节
- ✏️ **提出修改意见**：用户输入修改意见，AI 修改后再次审核该章节
- 🔄 **重新生成**：重新生成该章节内容

### 可选章节额外选项
- ⏭️ **跳过章节**：跳过当前章节（仅限第 11 章附录）

## 使用说明

### AI 使用指南

1. **按章节顺序询问**：严格按照第 1-11 章的顺序进行
2. **提供参考信息**：每次询问时都要提供相关的参考信息
3. **生成章节内容**：根据用户输入和模板格式生成章节内容
4. **章节审核**：生成后展示内容，提供审核选项
5. **处理审核结果**：
   - 确认通过：继续下一章节
   - 提出修改意见：修改后再次审核
   - 重新生成：重新生成该章节
   - 跳过章节：跳过当前章节（可选章节）

### 用户使用指南

1. **参考提示**：每次询问时都会提供参考信息，帮助填写
2. **灵活填写**：可以根据实际情况调整内容
3. **章节审核**：每个章节生成后都可以审核和修改
4. **可选章节**：第 11 章附录可以跳过
